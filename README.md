# dotfiles
zsh use ohmyzsh theme.

# Install dependency
```
pacman -S bspwm sxhkd polybar rxvt-unicode neovim neofetch dunst mpv mpd ncmpcpp picom rofi zathura ranger xorg xorg-server feh git
```
# Install yay
```
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
```
